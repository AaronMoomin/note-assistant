## Change Log

### [ 0.5.0 ] - 2020-10-22

#### Features

* 增加 注册功能
* 增加 登录注册动画
* 增加 登录页面手机号输入框清空数据按钮

#### Changed

* 变更 登录页面密码验证触发方式
